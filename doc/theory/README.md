# 理論的背景

このディレクトリには、SNI Visualizer 実装の基礎となる理論的背景、形式的定義、および証明が含まれます。

## 目的
本プロジェクトのコードは、これらのドキュメントで説明されている理論の **実装** です。コードは「どのように動くか」の正本ですが、これらのドキュメントは「なぜそのように設計されているか」「何を保証するのか」の正本となります。

## 内容
- **準備中**: 関連する Google ドキュメント（理論定義など）をエクスポートし、ここに配置する予定です。

## プロジェクト構造
- `doc/architecture.md`: システムコンポーネントの責務と境界。
- `doc/project.md`: プロジェクトの概要とエントリポイント。
- `sni-engine/`: コアロジックの実装。
